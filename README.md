# 31corefs

31corefs is a modern, cross-platform filesystem written in Rust. It support advanced features like subvolume management, snapshot and CoW.

## Features

* Copy on Write
* Sparse file
* Subvolume
* Snapshot

## lib31corefs

Core library to access 31corefs, without any platform-related code.
